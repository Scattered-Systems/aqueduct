# aqueduct

Aqueduct implements the recently formalized tweakable sleeve, a novel sleeve achieving modularity by implementing a composition mechanism based on tweakble hash functions.

## Getting Started

### _Import the crate_

    aqueduct = "0.1.0"

### _Basic Usage_

  use aqueduct
  
  fn main() {
    println!("Welcome to aqueduct");
  }
